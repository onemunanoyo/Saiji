# 塞耳 Saiji（さいじ）

> YouTube・Twitch のライブチャットから、不快なコメントを静かに消す Chrome 拡張機能

## どんな拡張機能？

ライブ配信のチャット欄には、Bot フィルターをすり抜けた攻撃的なコメントが流れることがあります。  
**塞耳 Saiji** は、あなたが登録した NGワードを含むコメントをブラウザ側で自動的に非表示にします。

- サーバーへの通信なし・アカウント連携なし
- データはすべてあなたのブラウザにのみ保存
- YouTube と Twitch に対応

## 使い方

### 1. NGワードを登録する

拡張機能のアイコンをクリック → **「設定・NGワード管理」** を開きます。

```
入力欄にワードを入力 → Enter または「追加」ボタン
```

登録したワードを含むコメントは、チャット欄に表示される前に自動的に削除されます。

**例：**
| 登録ワード | 非表示になるコメント例 |
|---|---|
| `死ね` | 「死ね」「お前死ねよ」「死ねカス」 |
| `うざい` | 「うざい」「マジうざいんだけど」 |

> 判定は**完全一致（部分一致を含む）**です。「死ね」を登録すると「死ねカス」も非表示になります。

---

### 2. フィルタを一時的に止める

ツールバーのアイコンをクリック → **「フィルタを OFF にする」** ボタン。

特定の配信でフィルタを無効にしたい場合などに使います。再度クリックで ON に戻ります。

---

### 3. 3回アウトの自動追放

同じユーザーが3回 NGワードに引っかかると、そのユーザーは**追放リスト**に追加されます。  
追放リストに登録されたユーザーのコメントは、内容に関わらず以降すべて非表示になります。

```
1回目の違反 → カウント +1（コメントは非表示）
2回目の違反 → カウント +2（コメントは非表示）
3回目の違反 → 追放リストに追加（以降、全コメント非表示）
```

> 違反カウントは **15分ごとに自動リセット**されます。「たまたま」一致してしまったユーザーへの誤判定を防ぐためです。

---

### 4. 追放リストを確認・解除する

設定ページの **「追放リスト」タブ** から確認できます。

- 一覧に表示されている author-id を確認
- 「解除」ボタンで個別に解除できます

> 追放リストは **7日ごとに自動でリセット**されます。永続的な BAN ではありません。

## 動作の仕組み

```
YouTube / Twitch のチャット欄
        ↓ 新しいコメントが追加される
  MutationObserver が検知
        ↓
  ① 追放リストに含まれるユーザー？ → YES → 即削除
        ↓ NO
  ② NGワードが含まれる？         → NO  → そのまま表示
        ↓ YES
  コメントを削除 + 違反カウント +1
        ↓
  3回に達した？ → YES → 追放リストへ登録
```

## データの保存について

| データ | 保存場所 | 有効期間 |
|---|---|---|
| NGワード | ブラウザのローカルストレージ | 手動削除まで永続 |
| 追放リスト | ブラウザのローカルストレージ | 7日ごとに自動リセット |
| 違反カウント | メモリ（揮発性） | 15分ごとに自動リセット |
| フィルタ ON/OFF | ブラウザのローカルストレージ | 手動変更まで永続 |

**外部サーバーへのデータ送信は一切行いません。**

### DOM セレクタの更新

YouTube / Twitch のサイト更新でチャット欄のセレクタが変わった場合は、  
`src/shared/selectors.ts` の1箇所だけ修正してください。

```typescript
export const SELECTORS = {
  youtube: {
    chatContainer: 'yt-live-chat-item-list-renderer #items', // ← ここ
    ...
  },
  twitch: {
    chatContainer: '.chat-scrollable-area__message-container', // ← ここ
    ...
  },
};
```

## よくある質問

**Q. 追放したユーザーは相手にバレますか？**  
A. バレません。非表示にするのはあなたの画面だけです。配信者や他の視聴者には影響しません。

**Q. NGワードは正規表現で指定できますか？**  
A. 現時点では文字列の部分一致のみです。正規表現対応は今後検討します。

**Q. Firefox でも使えますか？**  
A. 現時点では Chrome 系のみ対応しています。

**Q. 「死ね」と登録したら「必死ね」も消えますか？**  
A. はい、部分一致のため消えます。意図しない場合はより具体的なワードを登録してください。
